Projects

Clubbernaut App


Clubbernaut is a web application for UMass students to find clubs. It allows students to follow clubs and receive updates from said clubs as well as create clubs and send updates. There are many more feautes as well! It was built using Next Js, Node Js, and Supabase.

UMass Course Reviews


UMass Course Reviews was created in a hackathon with a partner to enhance the course selection process for students at the University of Massachusetts. Offering an innovative alternative to RateMyProfessor, this platform focuses on providing comprehensive details about courses, independent of the instructors.

MLB Attendance Web Scraper


This project uses Python and Pandas to scrape attendance data from baseball-reference.com. It allows the user to select which season(s) they want to receive data for and copys that data to a csv file for them.
