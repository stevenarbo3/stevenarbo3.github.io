    About
    Me

    Hello, I'm Steven Arbo. I am currently entering my senior year at the University of Massachusetts Amherst. I am studying computer science, minoring in mathematics, and focusing on data science in my personal projects. In my free time, I love designing new projects, working out, and sports.

    Data Job Analysis

    Used PostgreSQL to analyze job postings on various websites. Answered questions such as "what are the top paying data scientist skills?" and "what are the most optimal skills to learn for data science?"

    Predicting ops

    Used pybaseball library in python to extract batting stats for each player from 2000-present. Used pandas and sklearn to clean the data and build a model that can accurately predict a player's ops in the next season.

    Atlantic 10 Analysis

    Use pandas to extract Atlantic 10 team stats from last year from sports-reference.com. Cleaned the data, calculated advanced stats and analyzed how teams compared with each other in conference play.

    Spotify Analysis

    Use spotipy library in python to easily use spotify's api to extract data on my listening habits and analyze them. Also clean the data and answer questions such as "does track number influence a song's popularity?"

    MLB Attendance

    Use pandas to extract mlb attendance data from baseball-reference.com for every game from a specific season. Clean and convert the data to a csv.

    Clubbernaut App

    Clubbernaut is a web application for UMass students to find clubs. It allows students to follow clubs and receive updates from said clubs as well as create clubs and send updates. There are many more feautes as well! It was built using Next Js, Node Js, and Supabase.
