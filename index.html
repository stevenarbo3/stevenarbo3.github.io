    Job Posting Analysis

    Analyzes job postings on various websites. Answers questions such as "what are the top paying data scientist skills?" and "what are the most optimal skills to learn for data science?"

    Predicting ops

    Extracts batting stats for each MLB player from 2000-Present using the pybaseball library in python. Cleans the data using pandas and builds a model that can accurately predict a player's OPS (on base plus slugging percentage) in the next season using sklearn.

    Lisp interpreter

    This is a basic Scheme interpreter written in Python using PLY (Python Lex-Yacc). It can parse and evaluate simple Scheme expressions, including arithmetic operations, variable definitions, and conditionals. It is based on the interpreter written in this website https://norvig.com/lispy.html.

    Hooper App

    This is full-stack web app for users to log basketball stats and track performance over time with an interactive React + Vite frontend, FastAPI backend for fetching data, Clerk authentication, and OpenAi API to generate personalized NBA player comparisons based on user data. Frontend and backend dockerized for portability and easy deployment.

    Fantasy Football Draft Analysis

    Extracts fantasy league data using the Sleeper API and fantasy stats using pandas read_html from sports-reference.com. Cleans, joins, and analyzes data to see how league members draft.

    Clubbernaut App

    Clubbernaut is a web application for UMass students to find clubs. It allows students to follow clubs and receive updates from said clubs as well as create clubs and send updates. There are many more feautes as well! It was built using Next Js, Node Js, and Supabase.
